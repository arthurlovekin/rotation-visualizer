# Rotation Visualizer
This tool shows different representations of a rotation in 3D, including a visual representation and various algebraic representations. All of the representations are a "view" into the same underlying rotation, and update automatically as the user interacts with the sliders or text boxes.

## Controls
- Left-click and drag to rotate the object with respect to the world.
- Right-click and drag to rotate the camera with respect to the world.
- Mouse wheel to zoom in and out.

## Development
To run the development server, run the following, then open the browser to see the app (trunk serves on `http://127.0.0.1:8080/` by default).

```
trunk serve
```
